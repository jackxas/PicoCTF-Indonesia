Kita diberikan sebuah folder yang di zip

Masukkan folder tersebut ke linux,dan unzip folder tersebut (unzip challenge.zip)

Kita akan mendapatkan sebuah direktori baru

Pergi ke direktori tersebut lalu ls -la (menampilkan semua berkas dan folder di dalam direktori saat ini secara lengkap dan mendalam)

Terdapat file git didalam direktori tersebut (file yang mencatat setiap perubahan pada berkas proyek)

Jalankan perintah git log untuk melihat riwayat rekaman (commit history) dari sebuah repositori Git

Terdapat commit yang sudah dilakukan 

Jalankan perintah git checkout "namacommmit" untuk kembali ke masa lalu

Setelah itu buka file message.txt
