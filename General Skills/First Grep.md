## Pengerjaan
Disini kita diberikan sebuah file yang berisi teks acak

Dilinux ada command yang bernama grep yang berfungsi untuk mencari dan menyaring baris teks di dalam file atau output sistem yang cocok dengan pola (pattern) atau kata kunci tertentu

Untuk mencari kata di dalam sebuah file kita menggunakan sintaks dasar berikut:grep "kata_kunci" nama_file.txt

Di chalangge ini kita memasukkan perintah : grep 'picoCTF' file

Nanti akan menampilkan isi file,tetapi kata kunci yang kita cari akan berwarna merah
