## Deskripsi : 
A company stored a secret message on a server which got breached due to the admin using weakly hashed passwords. Can you gain access to the secret stored within the server?

## Pengerjaan

Tantangan ini berkonsep tentang hash

Hashing adalah proses matematis untuk mengubah data atau teks biasa (seperti kata sandi atau file) menjadi deretan karakter acak dengan panjang yang tetap, yang disebut sebagai hash.

Masuk ke server yang telah diberikan,disitu kita akan diberikan sebuah kalimat hashing dan kita disuruh menebak passwordnya

Disini saya menggunakan website https://crackstation.net/ untuk melakukan proses hashing

Masukkan kalimat hashing tersebut website crackstation dan website tersebut akan menebak passwordnya
